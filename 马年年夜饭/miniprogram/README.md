# 🐴 马年新春祝福小程序

## 📱 项目介绍

这是一个微信小程序版本的马年新春祝福平台，包含以下功能：
- ✅ 发送新春祝福卡片（6种精美模板）
- ✅ 上传年夜饭照片
- ✅ 年夜饭照片墙展示
- ✅ 实时统计数据
- ✅ 超级喜庆的马年主题界面

## 🎯 功能特点

### 1. 首页
- 📊 实时统计数据展示
- 🎨 特色功能介绍
- 💝 最新祝福预览
- 🏮 喜庆装饰动画

### 2. 发送祝福
- 🎨 6种精美模板：龙马精神、红红火火、金玉满堂、春风得意、国潮风、水墨丹青
- ✍️ 自定义祝福语
- 👤 填写昵称和祝福对象
- 👁️ 实时预览祝福卡片

### 3. 上传照片
- 📸 支持1-3张照片上传
- 📝 添加新年寄语
- 🔒 隐私设置（公开/私密）
- 💾 本地存储

### 4. 照片墙
- 🖼️ 瀑布流展示年夜饭照片
- 👥 显示上传者信息
- ⏰ 时间显示（刚刚、X分钟前等）
- 🔍 点击预览大图

## 🚀 如何使用

### 方法一：微信开发者工具（推荐）

1. **下载微信开发者工具**
   - 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 下载并安装对应系统的版本

2. **导入项目**
   - 打开微信开发者工具
   - 点击「导入项目」
   - 选择 `miniprogram` 目录
   - 填写项目名称：`马年新春祝福`
   - 点击「导入」

3. **运行项目**
   - 导入后自动编译运行
   - 在模拟器中预览效果
   - 可以点击「真机调试」在手机上测试

### 方法二：在微信中预览

1. 在微信开发者工具中点击「预览」
2. 使用微信扫描二维码
3. 在手机微信中查看效果

## ⚠️ 重要说明

### 关于 tabBar 图标

小程序需要 tabBar 图标图片，请按以下步骤添加：

1. 在 `miniprogram` 目录下创建 `images` 文件夹
2. 添加以下图标文件（尺寸：81px × 81px）：
   - `home.png` - 首页图标（未选中）
   - `home-active.png` - 首页图标（选中）
   - `greeting.png` - 祝福图标（未选中）
   - `greeting-active.png` - 祝福图标（选中）
   - `upload.png` - 上传图标（未选中）
   - `upload-active.png` - 上传图标（选中）
   - `gallery.png` - 照片墙图标（未选中）
   - `gallery-active.png` - 照片墙图标（选中）

**临时方案：**
如果暂时没有图标，可以修改 `app.json` 中的 tabBar 配置，删除 `iconPath` 和 `selectedIconPath` 字段，使用纯文字 tabBar。

### 数据存储

当前版本使用微信小程序的本地存储（`wx.setStorageSync`），数据仅保存在用户手机本地。

**如需多人共享数据，需要接入云开发：**

1. 开通微信云开发
2. 创建数据库集合（greetings、photos）
3. 修改 `utils/storage.js` 使用云数据库 API
