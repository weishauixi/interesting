    │   ├── home.js                   # 页面逻辑
    │   ├── home.wxss                 # 页面样式
    │   └── home.json                 # 页面配置
    ├── greeting/                     # 祝福页
    │   ├── greeting.wxml

## 🚀 如何运行

### 方法 1：使用纯文字 tabBar（最简单，无需图标）

1. 将 `app-noicon.json` 的内容复制到 `app.json`
2. 打开微信开发者工具
3. 导入 `miniprogram` 目录
4. 立即运行！

### 方法 2：添加自定义图标

1. 在 `miniprogram/` 下创建 `images` 文件夹
2. 添加 8 个图标文件（81x81px）：
   - home.png, home-active.png
   - greeting.png, greeting-active.png
   - upload.png, upload-active.png
   - gallery.png, gallery-active.png
3. 打开微信开发者工具导入项目

## 💾 数据存储说明

当前版本使用**微信小程序本地存储**：
- ✅ 无需配置服务器
- ✅ 数据永久保存（用户手机本地）
- ⚠️ 不同用户数据不共享
- ⚠️ 存储空间限制 10MB

**如需多人共享数据，需要接入微信云开发**

## 🎯 核心技术

- **框架**：微信小程序原生框架
- **语言**：WXML + WXSS + JavaScript
- **存储**：wx.setStorageSync / wx.getStorageSync
- **API**：
  - wx.chooseImage（选择图片）
  - wx.previewImage（预览图片）
  - wx.showToast（提示信息）
  - wx.navigateTo（页面跳转）
